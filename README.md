# [TeleMute](https://github.com/MutePuker/TeleMute)
*TG-CLI based broadcasting bot!*
# Install
```
cd $HOME
git clone https://github.com/MutePuker/TeleMute.git
cd TeleMute
chmod +x tg
./tg
# Enter a phone number & confirmation code.
./launchsh```
# Create a bot!
```
